# 📊 Adidas Sales Report

This project visualizes **Adidas sales data** using **Power BI**.

The report helps answer the following **business questions**:

* ✅ Key KPIs including Total Sales, Total Profit, and Units Sold
* ✅ Sales distribution by region
* ✅ Comparison of sales channels (Online, In-store, Outlet)
* ✅ Analysis of the top-performing cities
* ✅ Monthly sales and profit trends
* ✅ Sales performance by product category
* ✅ Year-over-year sales growth (%)
* ✅ Interactive analysis using filters (Year, Region, Product, Sales Method)

### Key focus areas during this project

* Clean and user-friendly dashboard design
* Effective presentation of KPIs
* Data visualization following storytelling principles
* Interactive filtering for an enhanced user experience

## 🛠️ Technologies Used

* **Power BI**
* **DAX**
* **Power Query**
